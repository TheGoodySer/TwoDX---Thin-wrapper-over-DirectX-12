# TwoDX---Thin-wrapper-over-DirectX-12

My first learning project: a thin wrapper over DirectX 12 that eliminates boilerplate code.



\# Requirements

* Windows 10/11 SDK
* Visual Studio 2022 or newer with C++ components
* DirectX Tool Kit 12 Desktop Windows 10 (name for NuGet Package Manager: directxtk12\_desktop\_win10)
* DirectX 12 Help Headers (name for NuGet Package Manager: Microsoft.Direct3D.D3D12)
